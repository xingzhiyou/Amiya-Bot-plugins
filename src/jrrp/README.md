查询每日人品运势，支持今日、本周、本月和历史平均人品查询。

## 功能说明

- **今日人品**：查询今日幸运指数和运势等级
- **本周人品**：查询本周平均幸运指数
- **本月人品**：查询本月平均幸运指数
- **历史平均**：查询所有历史记录的幸运指数

## 使用方式

- 查询今日人品
    - 发送 `jrrp` 或 `今日人品` 或 `今日运势`

- 查询本周平均人品
    - 发送 `weekjrrp` 或 `本周人品` 或 `本周运势` 或 `周运势`

- 查询本月平均人品
    - 发送 `monthjrrp` 或 `本月人品` 或 `本月运势` 或 `月运势`

- 查询历史平均人品
    - 发送 `alljrrp` 或 `总人品` 或 `平均人品` 或 `平均运势`

## 配置说明

插件支持通过配置文件自定义运势等级范围：

- `ranges`：运势等级范围配置（超吉、大吉、吉等）
